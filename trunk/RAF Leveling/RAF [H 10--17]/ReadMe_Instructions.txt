
There are numerous Popup windows used throughout these profiles
The popup windows are used to synchronize all of the characters in the party
Please read the popups and follow the instructions they provide

The RAF characters will automatically detect when they need to advance to the next profile
Before entering the instance each time, the RAF characters will run a level check. 
If they meet the level requirements, a pop up window will alert you to stop the bot and advance to the next profile
 
At that time, the user will need to:
1. Stop the bot on the Main Character and RAF Characters
2. Manually exit the instance the Main Character is currently in. Do not move the Main character after exiting the instance
3. Click load profile on the bot
4. Browse to the Main Character folder and select the next profile in the sequence for the Main Character, and click start on the bot

5. Do the same for the RAF characters, be sure to select the correct profile from the RAF Character folder

				******* Instructions *********

Log into WoW on all the characters
Using the main Character, invite all the RAF characters to a party
The Main Character must be the leader of the party

******* Main Character	*********

1. Put the bot using Questing mode
2. Determine the current level of the RAF characters you are running.
3. Click the Load Profile button on the bot, browse to the Main Character folder
4. Select the correct profile, based on the level of the RAF characters you are running
5. Click start on the bot

******* RAF Characters	*********

1. Put the bot using Questing mode
2. Determine the current level of the RAF character
3. Click the Load Profile button on the bot, browse to the RAF Power Leveling folder
4. Select the correct profile, based on the level of the RAF characters you are running
5. Click start on the bot
